=== install service ===
1. open cmd.exe and go the service folder
2. write following
   C:\Windows\Microsoft.NET\Framework\v4.0.30319\InstallUtil.exe JsonPubSubService.exe

=== uninstall service ===
1. open cmd.exe and go the service folder
2. write following
   C:\Windows\Microsoft.NET\Framework\v4.0.30319\InstallUtil.exe /u JsonPubSubService.exe
